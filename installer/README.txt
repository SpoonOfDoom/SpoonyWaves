SpoonyWaves - Audio Waveform Visualization Tool
===============================================

Thank you for installing SpoonyWaves!

WHAT IS SPOONYWAVES?
SpoonyWaves is a tool that creates waveform visualizations from your audio files.
Simply right-click any supported audio file and select 
"Generate Waveform with Spoony Waves" to create a visualization.

SUPPORTED FORMATS:
- MP3 (.mp3)
- WAV (.wav)
- FLAC (.flac)
- M4A (.m4a)
- AAC (.aac)

HOW TO USE:
1. Right-click on any supported audio file
2. Select "Generate Waveform with Spoony Waves" from the context menu
3. Wait for processing to complete
4. Find your waveform video in the same folder as the original audio file

COMMAND LINE USAGE:
You can also use SpoonyWaves from the command line:
SpoonyWaves.exe "path\to\your\audio\file.mp3"

TROUBLESHOOTING:
If you encounter any issues, try running the diagnostics tool:
SpoonyWaves.exe --diagnostics

UNINSTALLING:
To uninstall SpoonyWaves, use "Add or Remove Programs" in Windows Settings,
or use the uninstall shortcut in the Start Menu.
